# Express for REST api OPS-CORE


REQUIRED
```
node i nodemon        = "^2.0.20"   // live testing
node i body-parser    = "^1.20.1"   // For get body from method post
node i mysql          = "2.18.1"
express               = "^4.18.2"
```

IMPORTANT
```
npm install           // install all package
npm run ops           // run OPS-CORE default on port 3000
```
